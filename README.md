# probabilistic_robotics

2023年度 確率ロボティクス課題<br>
パーティクルフィルタによりノイズがのった制御指令値で動いているロボットの位置推定を行う．<br>
Jupyter Notebookを使用したパーティクルフィルタを使用した位置推定のシミュレーションを行える．<br>

## 確認済み実行環境
* Jupyter Notebook
* Python 3.8.10
* WSL2
* CPU: intel core-i7 13700k
